## enable_if를 사용하는 위치
1) 함수 리턴 타입
2) 함수 인자 타입 => 생성자에서 주로 사용
3) 템플릿 인자 => 함수 자체의 모양이 단순해 보이는 장점이 있다.

```c++
template<typename T>
typename enable_if<is_inegral<T>::value >::type
foo(T a)
{}


template<typename T>
void foo(T a,
typename enable_if<is_inegral<T>::value >::type* = nullptr)
{}

template<typename T,
         typename enable_if<is_inegral<T>::value >::type* = nullptr>
void foo(T a)
{}

```
